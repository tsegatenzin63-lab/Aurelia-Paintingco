# Deploying to Vercel

1. Install dependencies:
   ```
   npm install
   ```

2. Run locally:
   ```
   npm run dev
   ```

3. Deploy:
   - Create a GitHub repo and push the project
   - On Vercel, click "Import Project" -> select the GitHub repo -> Deploy
   - Add domain `www.aureliapainting.com.au` in Vercel dashboard and follow DNS instructions from your registrar.
